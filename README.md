# Global-Energy-Sustainability-Dashboard
An end-to-end Power BI project analyzing global renewable energy trends and carbon footprint (2000-2020).
Key Features:
		  	1.Interactive Map tracking renewable energy % globally.
        2.Advanced DAX measures for Carbon Intensity and Green Energy Growth.
        3.Star Schema data modeling (Fact and Dimension tables).

Tech Stack: Power BI Desktop, Power Query, DAX, Excel/CSV.
Data Source:global-data-on-sustainable-energy (1).csv
